# Dotfiles

A minimal, fast, and modern Linux setup focused on Wayland, performance, and clean aesthetics.

---

## Software Stack

| Category        | Tool        |
|----------------|------------|
| Compositor     | niri       |
| Shell          | fish       |
| Terminal       | kitty      |
| Prompt         | starship   |
| System Info    | fastfetch  |
| Navigation     | zoxide     |
| File Manager   | yazi       |
| Font           | JetBrains Mono Nerd Font *(or any Nerd Font)* |

---

## ⚙️ System Setup

This configuration is built on:

- **Distribution:** CachyOS  
- **Bootloader:** Limine  
- **Login Manager:** greetd + tuigreeter  
- **Desktop Environment:** niri + Noctalia-Shell  

---

## 📦 Dotfile Management (GNU Stow)

This setup uses **GNU Stow** to manage dotfiles via symlinks.
